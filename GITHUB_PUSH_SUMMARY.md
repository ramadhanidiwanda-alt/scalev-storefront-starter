## 🚀 Next Steps for Users

### 1. Clone & Setup
```bash
git clone https://github.com/ramadhanidiwanda-alt/scalev-storefront-starter.git
cd scalev-storefront-starter
npm install
cp .env.example .env.local
# Add Scalev credentials to .env.local
npm run dev
```

### 2. Get Scalev Credentials
- Sign up at https://dashboard.scalev.com
- Get Store ID and Storefront API Key
- Add to `.env.local`

### 3. Deploy to Vercel
- Connect GitHub repo
- Add environment variables
- Deploy
